Prin aceasta aplicatie am realizat mai multe interfete pentru gestionarea unei firme de productie/retail.
Prima interfata este cea de user, in care prin butonul „Register" se creează un nou user în baza de date, iar prin butonul „Login” se face logarea către pagina principală sau cea de admin.
Pentru accesarea paginii admin trebuiesc introduse username:eduard parola:cristescu, iar aceasta poate aduce date din baza de date,realiza si imprima, cu ajutorul acestora, grafice cu statistici ale comenzilor primite.
In interfata de user se pot adauga comenzi, se pot vedea comenzile pe care un user deja le-a realizat si se pot modifica sau sterge.
Nu in ultimul rand, userul poate să vadă detaliile contului său, sau, dacă are probleme, să se întoarcă în meniul principal prin butonul „Logout” și să trimită o sesizare către una dintre echipele ce se ocupă de site.
